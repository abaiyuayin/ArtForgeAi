## 1. Demo 简介

**是什么：** ArtForgeAI 是一个纯前端运行的独立游戏美术素材处理网站（Vue 3 + Vite + TypeScript + Tailwind CSS），所有数据保存在浏览器本地 IndexedDB，无需注册、无需会员、无需上传云端。

**面向谁：** 独立游戏开发者、像素 / 2D 游戏美术创作者、需要快速处理素材但不想安装重型软件或付费订阅的小团队。

**主要功能：**

- **图片处理**：支持网格 / 手动裁剪、AI 抠图（Flood Fill / Chroma Key / K-Means 聚类），可导出 PNG。
  - 【TODO：截图 1 - 图片处理 / 抠图界面截图】
- **序列帧工坊**：视频、GIF、图片序列一键转序列帧，支持裁剪、相似帧检测、预览播放，可导出视频 / GIF / 精灵图 ZIP（PNG + JSON）。
  - **新增精灵图裁剪**：上传一张精灵图，按列数行数自动裁剪为独立 PNG 帧图，实时网格预览。
  - 【TODO：截图 2 - 序列帧工坊 - 视频转序列帧界面截图】
- **帧编辑器**：抠图与水印去除采用可叠加流水线架构（抠图 → 水印擦除 → 输出），两种效果不互斥；支持 Ctrl+Z 撤销（最多 30 步）、边界框选、应用至所有选中帧。

## 2. Demo 创作思路

**灵感来源：** 我自己在开发独立游戏 Demo 时，经常需要在 PS、AE、在线转换工具、格式工厂之间来回切换，只为把一段视频转成序列帧、把 GIF 拆帧、或者给角色图抠个背景。这些工具要么收费、要么要上传云端，对小团队很不友好。

**想解决的问题：**

- 美术素材处理工具分散，学习成本高；
- 免费工具往往强制云端上传，存在隐私和版权风险；
- 序列帧、精灵图、抠图等 2D 游戏常用功能没有一站式的纯前端方案。

**为什么做这个方向：** 我判断独立游戏开发者最需要的不是"更华丽的 AI 生成"，而是"能把已有素材快速处理成可用格式"的轻量工具。所以 ArtForgeAI 选择扎根在**媒体处理 + 本地资源库**，AI 功能（如文生图）仅作为可选项且必须用户自配 API Key，绝不提供虚假占位功能。

**Step 4：帧编辑器流水线架构重写与精灵图裁剪**

在 TRAE 中完成了本轮关键架构升级与功能扩展：

- **抠图 + 水印流水线**：将原本互斥的"抠图模式"与"水印模式"重写为可叠加的流水线（原始图 → 抠图参数 → 水印区域擦除 → 输出）。无论先抠图后去水印、还是先去水印后抠图，两种效果都能正确叠加，"应用至所有帧"也能同步完整流水线。
- **跨会话状态持久化**：给每帧增加 `mattingParams` 和 `erasedRegions` 元数据，保存后重新打开编辑器、或多次"应用至所有帧"都能正确恢复叠加状态，不再丢失已处理效果。
- **边界框选修复**：水印框选改用 window 级鼠标事件监听，鼠标移出画布也能继续框选；坐标 clamp 到画布边界，确保边缘水印可被选中。
- **Ctrl+Z 撤销**：在滑块拖拽、颜色选择、模式切换、水印框选等操作前自动推入快照，支持最多 30 步撤销，恢复完整编辑器状态。
- **精灵图裁剪功能**：图片转序列帧新增精灵图裁剪模式，上传一张精灵图后按列数行数自动裁剪为独立 PNG 帧图，实时网格预览叠加青色网格线，裁剪后进入三步流程（上传裁剪 → 处理帧 → 导出）。
- **帧选择工具同步**：SHIFT + 点击复选框连续选取、复选框尺寸调大、隔帧选取按钮、删除未选帧 + 二次确认弹框，三项页面级修复同步到视频 / GIF / 图片三个转序列帧页面。

## 5. 开发心得（额外部分）

- **纯前端不等于简单**：视频解码、Canvas 像素操作、GIF 编码都在浏览器里跑，对兼容性和性能要求很高。TRAE 帮我快速定位了 H.265 视频黑屏、0×0 尺寸重试、相似帧全量对比等问题。
- **先跑通再重构**：早期所有功能都堆在 `WorkspaceView.vue` 里，虽然能跑但难以维护。在 TRAE 协助下按"修复 bug → 拆分页面 → 验证构建"的迭代节奏完成重构。
- **流水线思维解决叠加问题**：帧编辑器最初把抠图和水印设计成互斥模式，用户反馈"先抠图再去水印会丢抠图效果"后才意识到需要流水线架构。在 TRAE 协助下重构为独立追踪两个状态（`hasMatting` + `erasedRegions`）并在 `renderToCanvas` 中组合，配合每帧元数据持久化，彻底解决了跨会话状态丢失问题。
- **用户数据本地化**：IndexedDB + localStorage 的组合让项目完全开源、无后端，也符合独立开发者对素材隐私的敏感需求。
